Description

When we want to write a heading on our page, we use the <h1> element.


There should only be one <h1> per page, so if you want to have more than one heading, you can use <h2> after <h1>, <h3> after <h2>, and so on all the way to <h6> (maximum).


The <h1> element needs to be closed by an </h1> after you have written your custom heading.

Here is an example:

<h1>

This is my first heading

</h1>

<h2>

This is my second heading

</h2>

<h5>

This is my fifth heading

</h5>
