Image <img>

Web Html Css

<img src="image.png" alt="Image of a tree near a lake on a starry night">

img {
max-width: 100%;
border-radius: 2rem;
box-shadow: 10px 10px 20px red;
}
Description

When we want to place an image on our page, we use the <img> element.


You can use as many <img> elements as you want per page.


The <img> element does not need to be closed by a </img>.


To find and use the image we want, we need the src="" attribute inside the <img> element.


Here is an example:


<img src="image.png">


To make your images more accessible for the visually impaired, we add the alt="" attribute.


This way, the visually impaired can hear what the image presents for other people, and, if for some reason the image doesn't load, people can read what should have been there.


<img src="image.jpg" alt="Image of a tree near a lake on a starry night" >
