Description

When we want to place a table on our page, we use the <table> element.


You can use as many <table> elements as you want per page.


The <table> element needs to be closed by a </table>.


To make a row in your table, we use the <tr> (table row) element.
The <tr> element needs to be closed by a </tr>.

Inside this element you can make headings with the <th> (table header) element.
The <th> element needs to be closed by a </th>.

Inside the <tr> element you can also input data using the <td> (table data) element.
The <td> element needs to be closed by a </td>.


Here is an example:


<table>
<tr>
<th>
Heading
</th>
<td>
Data
</td>
</tr>
</table>
