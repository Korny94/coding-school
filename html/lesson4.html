Description

An "unordered list" is a list that has no specific order.

When we want to make an unordered list on our page, we use the <ul> element.

You can use as many <ul> elements as you want per page.

The <ul> element needs to be closed by an </ul> after you have written your custom list.

Every item on your list must be enclosed by the <li> element (list item). <li> Before the list item, and </li> after the list item.

Here is an example:

<ul>
<li>This is a list item</li>
<li>This is another list item</li>
</ul>

You can also choose other style types to decorate your unordered list. For example, square and circle (disc is default).

- Inside the ul tag (CSS), if you input "list-style-type: square;", you will get squares instead of bulletpoints (disc).

ul {
list-style-type: square;
}

- Inside the ul tag (CSS), if you input "list-style-type: circle;", you will get circles instead of bulletpoints (disc).

ul {
list-style-type: circle;
}
