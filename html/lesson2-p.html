Description

When we want to write a paragraph on our page, we use the <p> element.


You can use as many <p> elements as you want per page.


The <p> element needs to be closed by a </p> after you have written your custom paragraph.


Here is an example:


<p>

This is a paragraph that I have written

</p>

<p>

This is another paragraph. I really like to write paragraphs

</p>
