Description

When we want to make a link on our page, we use the <a> element. And inside the <a> element we put href="".


Inside the quotation marks is where we put our URL of choice.


You can use as many <a> elements as you want per page.


The <a> element needs to be closed by a </a> after you have written your custom hyperlink.


The text in between <a> and </a> is the text that will be clickable.


Here is an example:


<p>

This is a link to <a href="https://google.com">Google.com</a>

</p>

If you want the link to automatically be opened in a new tab, and not the default: Open in the same tab (target="_parent" or target="_self"), we use the "Target" attribute.


To open the link automatically in a new tab we put the attribute target="_blank" inside the <a> element.


<a href="https://google.com" target="_blank">Click here!</a>
