Description

When we want to group elements together, we can use the <div> element.


The <div> element needs to be closed by a </div>.


The <div> element acts as a container that you can use to group elements together.

You can also use the <div> element to customize the styling and layout.


Here is an example:


<div>
<h2>
This header..
</h2>
<p>

..and this paragraph, will be grouped together
</p>
</div>
