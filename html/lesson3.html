Description

An "ordered list" is a list that has a specific order.

When we want to make an ordered list on our page, we use the <ol> element.

You can use as many <ol> elements as you want per page.

The <ol> element needs to be closed by an </ol> after you have written your custom list.

Every item on your list must be enclosed by the <li> element (list item). <li> Before the list item, and </li> after the list item.

Here is an example:

<ol>
<li>This is list item number 1</li>
<li>This is list item number 2</li>
</ol>

You can also choose other variants to order your list. For example, roman numerals or letters (numbers are default).

- Inside the <ol>, if you input "type="i"", you will get lowercase roman numerals.

<ol type="i">

- Inside the <ol>, if you input "type="I"", you will get uppercase roman numerals.

<ol type="I">

- Inside the <ol>, if you input "type="a"", you will get lowercase letters.

<ol type="a">

- Inside the <ol>, if you input "type="A"", you will get uppercase letters.

<ol type="A">
