Description

When we want to edit a part of a text, we use the <span> element.


You can use as many <span> elements as you want per page.


The <span> element needs to be closed by a </span> after you have written the text you want to edit.


Here is an example:


<p>

This is a paragraph that I have written, but I need to edit <span>this part of the text</span>

</p>


<p>

This is another paragraph. But this time I need to edit <span>this part of the text</span>

</p>
